计时攻击基础：RSA算法及其实现

July 09, 2023 2 mins to read Jiaangk posts

RSA#

Generation#

  1. 取两个大质数 $p$ 和 $q$ ，为了安全性 $p$ 和 $q$ 长度应 大于 $2048$ 位；
  2. 设 $n = pq$；
  3. 设 $m = \phi(n) = (p-1)(q-1)$；
  4. 取 $e$，使得 $2 < e < m$且 $gcd(e, m) = 1$；
  5. 取 $d$，使得 $d \times e \equiv 1 \ (mod\ m)$；
  6. 此时，公钥为 $(n, e)$，私钥为 $(n, d)$。

Encryption#

约定 $x$ 为 plaintext， $y$ 为 cyphertext。

$$ y = x^e \ mod \ n $$

Decryption#

$$ x = y^d\ mod \ n $$

Proof#

当 $x$ 与 $n$ 互质时，有

$$ x = y^d \ mod \ n \ x = x^{ed} \ mod \ n $$

设 $k$ 为一自然数，则由同余的定义，有

$$ x = x^{km + 1} \ mod \ n \ = x \times (x^{\phi(n)})^k \ mod \ n $$

由欧拉定理，得

$$ x = x \ mod \ n = x $$

当 $x$ 与 $n$ 不互质时，不妨设 $x$ 为 $p$ 的倍数，记为 $x = ap$，其中 $a$ 是一正整数。

$$ x^{\phi(q)} \equiv 1 \ (mod \ q) \ x^{k\phi(p)\phi(q)} \equiv 1 \ (mod \ q) \ x^{k\phi(p)\phi(q)} = 1 + bq $$

则

$$ x = x^{km+1} \ mod \ n \ = x \times (1 + bq) \ mod \ n \ = (x + bqx) \ mod \ n \ = x \ mod\ n \ = x $$

Security#

若想通过公钥推知私钥，则必须对密钥 $n$ 进行分解质因数，由于这是一个数论难题，因此只需要将 $n$ 设置的足够大，则可以确保安全性。

RSA-CRT#

由于接收者同时知道 $q$ 和 $p$ ，故由中国剩余定理(CRT)，将 $mod \ n$的操作拆分，可实现加速。

Decryption#

$$ m_1 = y^d \ mod \ q \ m_2 = y^d \ mod \ p $$

通过预处理出 $(d \ mod \ q)$ 和 $(d \ mod \ p)$，将上式转化为

$$ m_1 = y^{d \ mod \ q} \ mod \ q \ m_2 = y^{d \ mod \ p} \ mod \ p $$

则此时原文

$$ x = m_2 + q \times (p^{-1} \times (m_1 - m_2) \ mod \ q) $$
